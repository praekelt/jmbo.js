Jmbo.js Docs

Jmbo.js is library built on top of Backbone.js that aims to improve app development by providing some standard and easily stylable modules.

Stacks

A stack manages hierchical and historical navigation by allowing you to push or pop view controllers on and off the stack.

Transitions between view controllers can be animated.

Constructor

Create a new instance of the StackControllerView and render it into an element.


                // Create an instance of the stack and render it into an element.
                stackController = new jmbo.ui.stack.ControllerView;
                $('#viewport').html(stackController.render());

                // Create a new `ControllerView`
                var newController = new jmbo.ui.view.ControllerView({
                    view: '#template-menu',
                    title: 'Menu'
                });

                // Push an initial element on to the stack.
                stackController.push(newController, {animation: false});

We've stored our menu template in a <script type="text/template"> tag. Script tags that aren't text/javascript are ignored by the browser. Making them perfect vessels for template storage.

Push

Pushing a view onto the stack controller is usually initiated by user interaction. (Pretend that this code below is attached to a click event) 


                // Create a Backbone view.
                var View = Backbone.View.extend({
                    render: function() {
                        this.$el.html('I am a view # ' + stackController.collection.length);
                        return this.el
                    }
                });
                // Notice that we pushed a Backbone View instead of an element. The 
                // `ControllerView` accepts both.
                var newController = new jmbo.ui.view.ControllerView({
                    view: new View, // where object becomes instance.
                    title: 'New View'
                })

                // The default animation is slide-left/right.
                stackController.push(newController);

Pop

Popping a controller view off the stack will return the popped controller view.


                oldControllerView = stackController.pop();

Tabs

Tabs provide radio-like controls that are used to switch between different ControllerViews.


                // Create the tab, and render it into an element
                tabController = new jmbo.ui.tab.ControllerView;
                $('#viewport').html(tabController.render());

                // create some controller views
                var menuController = new jmbo.ui.view.ControllerView({
                    view: '#template-menu',
                    title: 'Menu'
                });
                var lipsumViewController = new jmbo.ui.view.ControllerView({
                    view: '#template-lipsum',
                    title: 'Lorem Ipsum'
                });

                // And set them in the tab bar.
                tabController.set(menuController, lipsumViewController);
